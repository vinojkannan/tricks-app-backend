# tricks-app


### Plugins Used
```
python3
pipenv
flask-restplus
flask-sqlalchemy
```

### Commands used to setup this app locally
```
pipenv --help (will provide you for information)
pipenv shell (to enter into virtual space)
pipenv install
python manage.py
deactivate (to come out of virtual space)
```
